자주묻는 질문

Next.js는 React 기반의 프레임워크로 SSR과 SSG를 지원합니다.

Next.js는 서버 사이드 렌더링(SSR), 정적 사이트 생성(SSG), API 라우팅 등을 제공합니다.

React는 라이브러리이고 Next.js는 React를 기반으로 한 풀스택 프레임워크입니다.
